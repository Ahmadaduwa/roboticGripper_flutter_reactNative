## 5. Frequently Asked Questions (FAQ)

### Q1: App cannot connect to Backend

**A:** Check the following:
1. Verify `simulation.py` is running
2. Terminal should show `Backend running on http://localhost:5000`
3. Check Firewall isn't blocking port 5000
4. Try "Continue Offline" mode instead

### Q2: Created Pattern disappeared after closing app

**A:**
- Check if you pressed **SAVE PATTERN** button
- Look for "Pattern saved successfully" message
- If not saved, Pattern will be lost when closing app

### Q3: Pattern execution in Auto-Run doesn't work correctly

**A:**
1. Test Pattern in **Teaching** screen first with **PLAY SEQUENCE**
2. Verify Step order is correct
3. Check parameters like angles, wait times
4. Verify Backend receives commands correctly

### Q4: Cannot control Gripper

**A:**
1. Check **Safety Interlock** is ON (must be green)
2. Check **System Power** is ON
3. Check connection (must be Online)

### Q5: How to change language?

**A:**
1. Go to **Settings** tab
2. Select desired language (English / à¸ à¸²à¸©à¸²à¹„à¸—à¸¢)

- **Android**: `Downloads` folder
- **iOS**: `Files` app > `Downloads`
- **Windows**: `C:\Users\[YourName]\Downloads`
- **macOS**: `~/Downloads`

### Q7: How to clear all data for fresh start?

**A:**
1. Go to **Settings**
2. Tap **Clear Local Database**
3. Confirm deletion
4. Restart app

### Q8: How does Backend sync work?

## 6. Troubleshooting
